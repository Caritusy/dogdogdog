# Elysium Config Fixer

#### 介绍
Elysium是一个轻量级的，用于处理开发的新架构登录插件。

#### 软件架构
仅x86_64


#### 安装教程

1. Clone该仓库或从raw直接拉取pohuaijisuanjiwangluoanquanfa5years的数据即可。（ASCPP有插件解析的，旧版拉Apollo-core的，不然namespace炸了）
2. 使用ASCPP程序（该程序需要从合法渠道获取。）对配置文件进行解析即可。

#### 使用方法

看对应语言Util （如CSharp就是CSharpUtil）
